## Migration Steps between source cloud and target cloud

**Step 1:** Provision AWS/ECS on Fargate

in https://gitlab.fhnw.ch/spd/module/pcls/hs25/hs25-group06/iac-aws#setup[iac-aws]

- Follow the https://gitlab.fhnw.ch/spd/module/pcls/hs25/hs25-group06/iac-aws#setup[Setup steps] to configure the IaC setup.

- Test the output of the Infrastructure by running

```bash
tofu plan
```

- Then provision the Infrastructure by running

```bash
tofu apply
```


- Test if everything is running as expected in the https://eu-central-1.console.aws.amazon.com/ecs/v2/clusters?region=eu-central-1[AWS console]
- Test if the https://us-east-1.console.aws.amazon.com/route53/v2/hostedzones?region=eu-central-1#[DNS Entry] now points to AWS.

**Step 2:** Shutdown SwitchEngine Infrastructure

in https://gitlab.fhnw.ch/spd/module/pcls/hs25/hs25-group06/iac-infra[iac-infra]

- Shutdown the SwitchEngine Infrastructure

```bash
tofu destroy
```
